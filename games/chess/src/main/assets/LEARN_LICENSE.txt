Learn mode lesson data
======================

The chess "Learn" lessons bundled in `learn.json` are derived from the Lichess
open-source project "lila":

    https://github.com/lichess-org/lila  (ui/learn, translation/source)

this app that includes `learn.json` must comply with the AGPL-3.0, including
making the corresponding source available.
